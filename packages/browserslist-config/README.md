# Browserslist Config

Shareable browserslist configuration.

## Install

```bash
# install with yarn
$ yarn add -D @wayofdev/browserslist-config

# install with pnpm
$ pnpm add -Dw @wayofdev/browserslist-config
```

## Usage

**`.browserslistrc`**

```bash
extends @wayofdev/browserslist-config
```
